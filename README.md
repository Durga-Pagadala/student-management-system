# Student Management System

A Full Stack Student Management System built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- Add Student Details
- View Student Records
- Store Data in MongoDB
- REST API Integration
- Frontend and Backend Connection
- Responsive User Interface

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
