# 🎓 Student Management System

A Full Stack Student Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. This project allows users to add and manage student details through a frontend interface connected to a backend server and database.

---

# 📌 Project Overview

The Student Management System is a beginner-friendly full stack web application that demonstrates the integration of frontend, backend, and database technologies.

The application allows users to:
- Add student information
- Store student records in MongoDB
- Retrieve and display student data dynamically
- Connect frontend forms with backend APIs

This project was developed as part of a Full Stack Development Internship task.

# ⚙️ Installation and Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Durga-Pagadala/student-management-system.git
```

---

## 2️⃣ Open Project Folder

```bash
cd student-management-system
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Backend Server

```bash
node server.js
```

Server Output:

```bash
MongoDB Connected Successfully
Server running on port 3000
```

---

## 5️⃣ Run Frontend

Open:

```bash
index.html
```

using browser or local server.
